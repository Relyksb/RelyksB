## 🔬 US Patent Holder

**[US20210103808A1](https://patents.google.com/patent/US20210103808A1/en) — Automatic Triaging of Network Events**
*Filed: October 2019 | Inventor: Skyler Butler | Assignee: InteliSecure (acq. Proofpoint)*

Patented a machine learning system that automatically classifies and triages data loss prevention (DLP) incidents using predictive modeling against data sensitivity profiles — reducing analyst workload and improving response accuracy at scale. The system identifies similar incidents based on attribute profiles and applies ML classification to determine triage outcomes without human review.

## 🛡️ OWASP Contributor — MCP Security

Active contributor to the **OWASP MCP Top 10** (Phase 3 Beta), introducing three new security concepts:

| Term | Definition |
|---|---|
| **Agentic Intent Validation (AIV)** | Pre-execution semantic validation of MCP requests against organizational intent policies — the missing layer between auth and execution |
